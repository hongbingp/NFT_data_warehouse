# NFT_marketplace
Author: Hongbing Pan (hp2608)
## 1 Pre-requiresite
Node.js

Truffle

Ganache


## 2 Truffle setup
```
npm install -g truffle
npm install @openzeppelin/contracts
```

## 3 Deploying Contracts
```
$ truffle compile
$ truffle deploy
```

## 4 Testing contracts
```
$ truffle test test_name.js
```
